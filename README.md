Анотація до диплому

І. Стосовно вибору макету
Багато варіантів макетів передивилась із тих, що знаходяться у вільному доступі, було вибрано декілька варіантів для можливої розробки, але зупинилась саме на цій версії. 
Як на мене сайт середньої тяжкості (як було оговорено в умовах виконання диплому), нажаль макет без мобільної версії, без адаптивних картинок, без додатковий сторінок, але візуально досить приваблий (не надокучливий). 

ІІ. Стосовно співпраці з GitHub: 
- на початку роботи створено окремий репозиторій GitHub  для даного проекту, також для моделювання робочої ситуації  в ньому створено гілки stage, develop та гілки-feature на кожен блок; 
- після виконання відповідного блоку проект транспортувався в гілку develop, потім в гілку main. 

ІІІ. Стосовно прийнятих рішень: 
1)	Для кнопок «Shop all products» використала анімаційний hover-ефект з ресурсу HOVER.CSS. 
Так як стиль макету досить мінімалістичний – анімаційні ефекти використані в мінімальній кількості для гармонічного поєднання.
2)	Кнопочки вибору товарів підсвічуються легкою тінню при натисканні. 
3)	Footer - примінила form із bootstrap.
4)	Header – примінила адаптивне бургер-меню із bootstrap.

ІV.  Адаптив. 
Для сторінки примінено адаптив з використанням медіа-запитів.

-проведена оптимізація картинок (2 рази);
-пройдена процедура валідації для HTML, CSS;
-пройдена процедура компресування.
